# Сайт «Квест Ловушка» — MVP

## Что внутри

- `index.html` — главная страница сайта.
- `config.js` — цены, контакты, расписание, закрытые даты.
- `assets/styles.css` — дизайн.
- `assets/app.js` — календарь, слоты, форма бронирования.
- `apps-script/Code.gs` — код Google Apps Script для заявок в Google Таблицу и Telegram.

## Что уже работает

- адаптивная главная страница;
- динамический календарь на 14 дней вперёд;
- цены по времени;
- форма заявки;
- демо-режим без отправки заявки;
- готовая интеграция с Google Таблицей через Apps Script.

## Где менять цены

Откройте `config.js`:

```js
prices: [
  { label: 'Утро / день', from: '09:30', to: '15:30', price: 1500 },
  { label: 'Вечер', from: '17:00', to: '20:00', price: 2000 },
  { label: 'Поздний вечер', from: '21:30', to: '23:00', price: 2500 }
]
```

Меняете значения `price` — сайт показывает новые цены.

## Где менять расписание

В `config.js`:

```js
slots: ['09:30', '11:00', '12:30', '14:00', '15:30', '17:00', '18:30', '20:00', '21:30', '23:00']
```

## Как закрыть дату или слот

В `config.js`:

```js
closedDates: ['2026-05-20'],
closedSlots: ['2026-05-21|18:30']
```

## Как подключить заявки в Google Таблицу

1. Создайте Google Таблицу.
2. Откройте: `Расширения → Apps Script`.
3. Вставьте код из `apps-script/Code.gs`.
4. Сохраните.
5. Нажмите `Deploy → New deployment`.
6. Тип: `Web app`.
7. Execute as: `Me`.
8. Who has access: `Anyone`.
9. Скопируйте Web App URL.
10. Вставьте его в `config.js`:

```js
webhookUrl: 'ВАШ_URL_ОТ_APPS_SCRIPT'
```

## Как подключить Telegram

В Apps Script откройте `Project Settings → Script Properties` и добавьте:

```text
TELEGRAM_BOT_TOKEN = токен вашего бота
TELEGRAM_CHAT_ID = id вашего чата или группы
SHEET_NAME = Заявки
```
